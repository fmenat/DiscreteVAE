# DiscreteVAE
Experiments with Discrete VAE

* Using Gumbel-Max trick (a soft version)
* Binary VAE: Hashing (Done)
	* Good results show that Binary VAE outperforms Traditional VAE on Hashing
* Categorical VAE: Classes
